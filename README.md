# PortScan
How to use:
### Step 1

<h4>If you do not have the requirements installed, do the following:</h4>

```py
pip install -r requirements.txt
```
### Step 2
```py
python3 portScan.py -i <IP>
```
```sh
  ❯ python3 portScan.py -i 192.168.1.25
    [+] Host: 192.168.1.25
[◒] Scanning: 
    Discovering port 65535
    [+] Port 21 : OPEN
    [+] Port 22 : OPEN
```
